## 前置条件

- Node.js
- Yarn
- Web Driver（请参考：https://www.selenium.dev/documentation/webdriver/getting_started/install_drivers/ ）

## 准备工作

1. 在此目录下运行 `yarn` 命令
2. 打开 `config.js` 文件进行配置

## 运行

在此目录下使用命令 `yarn start`
